# Elastic Search 安装

## 1 实验背景

Centos 8

jdk-8-221

## 2 安装

### 2.1 下载

```
wget https://artifacts.elastic.co/downloads/elasticsearch/elasticsearch-6.6.0.tar.gzhttps://artifacts.elastic.co/downloads/elasticsearch/elasticsearch-6.6.0.tar.gz
```

### 2.2 解压

```
tar -xvzf elasticsearch-6.6.0.tar.gz
```

## 3 运行

```
./elasticsearch
```
